Project to display images in excel by coloring cells.

instructions

1)
place the jpg image in images folder with filename im.jpg

2)
run matlab script convertIm.m

3)
import csv file from csvImages into the excel spreadsheet disp.xlsm.
by default excel adds headers to each collumn to get around this, open the csv file with excel then copy and paste the data into disp.xlsm.

4)
Run the colourIm macro.
